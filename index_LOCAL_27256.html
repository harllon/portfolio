Projects

Color App

Intelitec Solutions

Esse aplicativo permite que o usuário visualize cores que podem ser escolhidas de três formas diferentes. A primeira consiste em escolher as componentes RGB da cor através de siliders, a segunda consiste em escrever as componentes RGB em hexadecimal e, a última consiste em escolher as componentes HSV através de sliders. Além disso, o usuário pode salvar as cores que gostar e visualizar informações sobre ela posteriormente. Os principais aprendizados no desenvolvimento desse aplicativo foram: Uso da recycler view para mostrar as cores salvas e, tornar os elementos da recycler view selecionáveis. Uso da room database para salvar as informações das cores que os usuários decidem salvar. ViewPager para poder criar as abas. Dialog Boxes para criar os modais que aparecem ao selecionar a cor, ao selecionar o FAQ e ao tentar se registrar. Firebase para poder realizar o cadastro e o login dos usuários e livedata para poder sincronizar os valores nas mudanças das cores. Por último e não menos importante, a linguagem kotlin foi treinada e utilizada na confecção deste aplicativo.

Sensor App

Aplicativo que permite recolher dados de diversos sensores disponíveis nos celulares como por exemplo acelerômetro, giroscópio. O usuário pode selecionar vários sensores de uma lista de opções, além de pode selecionar alguns métodos. Ao terminar as seleções e iniciar a coleta de dados, os sensores selecionados serão ativados e, ao fim da coleta caso o usuário queira salvar os dados, eles serão escritos em arquivos .csv, sendo um arquivo para cada sensor e metodo selecionado. O usuário também pode compartilhar os arquivos via email, sendo que os arquivos salvos serão automaticamente anexados ao corpo do email. Os principais aprendizados no desenvolvimento desse aplicativo foram: Room database para login, cadastro e salvar os dados provenientes dos sensores. O uso da biblioteca de sensores Android também foi praticado com esse aplicativo como acesso ao acelerômetro, giroscópio e outros sensores. Foi aprendido também como criar arquivos usando o aplciativo, nesse caso arquivos .csv, e, como exportar arquivos por diversos aplicativos anexando automaticamente os arquivos que se deseja compartilhar. A linguagem java foi treinada e utilizada para confecção deste aplicativo.
